# Genre Manager

The Genre Manager handles genre assignment logic, including dominant genre calculation for artists based on their track library.

## Overview

`GenreManager` is responsible for:

- Calculating dominant genres per artist from track data
- Managing genre mappings and normalization
- Applying genre updates to tracks via AppleScript
